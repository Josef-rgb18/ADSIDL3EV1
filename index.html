¿Que es HTML?

El codigo HTML(Lenguaje de marcado tiene por finalidad estrucuturar la pagina web y desplegar su contenido, para ello este se compone de etiquetas que conforman elementos, de los cuales los mas comunes son:

  • Encabezados
  • Parrafos
  • Listas
  • Imagenes
  • Vinculos
  • Etc...

Estructura de programación

  1. Párrafos

    Se crean con la etiqueta <p> y se utilizan para organizar y desplegar bloques de texto en la página.

  2. Listas

    Permiten agrupar elementos de forma ordenada (<ol>) o desordenada (<ul>) usando viñetas o números.

  3. Imágenes

    Se insertan con la etiqueta <img> para mostrar contenido visual. Requieren el atributo src para la ruta del archivo.

  4. Vínculos

    Son enlaces creados con la etiqueta <a> que permiten al usuario navegar hacia otras páginas web o secciones.
